# Code of Conduct

Tiramisu is a small, friendly project. The expectations are simple:

- **Be respectful.** Disagree with ideas; never with people.
- **Stay on topic.** Issues and PRs are for Tiramisu — image-editing, the
  creator workflow, the Mac app, the codebase. Not the broader culture wars.
- **Assume good intent.** Most miscommunication is wording, not malice.
- **No harassment, no doxing, no targeted personal attacks.** Of any kind.

If something feels off, email **hanley@taiso.ai** or open a private security
advisory. Reports are handled confidentially. Repeated or severe violations
result in being blocked from the project.

This is intentionally short. If you want a longer version with formal
enforcement procedures, the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
is a good drop-in — we'll adopt it formally if the project grows past the
point where the short version covers it.
